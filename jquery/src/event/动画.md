### 自定义动画实现
animate(params,speed,fun);

累加、累减动画

```
$("#btn").click(function(){
	$("#pos").animate({left:"+=20px"},fast);
})

```
同时执行多个动画

```
$("#pos").animate({left:"+=20px",height:"+=20px"},fast);

```
停止动画

```
stop()方法：停止阶段性动画。
stop(true)方法：停止当前动画后的所有阶段动画。

```
判断元素是否处于动画状态

```
$("#ele").is(":animated")
说明：
动画是顺序执行的。

```
